# مشروع إخفاء البيانات (Image Steganography - LSB)
### إعداد الطالب: محمد (هندسة الشبكات)

## 1. فكرة المشروع (Introduction)
يهدف هذا المشروع إلى إخفاء رسالة نصية سرية داخل صورة رقمية باستخدام تقنية البت الأقل أهمية (Least Significant Bit)، بحيث لا تتأثر جودة الصورة ولا يلاحظ المشاهد وجود أي تغيير.

## 2. الخطوات التقنية المتبعة (Implementation Steps)
تم تنفيذ المشروع وفق المعايير المطلوبة في المساق:
* **تحليل الصورة:** تم استخدام مكتبة `NumPy` لتحويل الصورة إلى مصفوفة بيانات خام (Raw Data Matrix).
* **معالجة البيانات:** يتم تحويل النص المطلوب إخفاؤه إلى سلسلة بتات (Bitstream).
* **التعديل (Encoding):** تم استخدام عملية `AND` لتصفير البت الأخير، ثم عملية `OR` لدمج بت الرسالة داخل البكسل.
* **علامة النهاية:** أضفت علامة توقف `_END_` في نهاية الرسالة لضمان استرجاعها بدقة عند فك التشفير.

## 3. المتطلبات (Requirements)
* Python 3.x
* Libraries: `PIL` (Pillow), `NumPy`

## 4. النتائج (Results)
* تم استرجاع النص بنسبة نجاح 100% دون أخطاء.
* الصورة الناتجة (Stego Image) مطابقة تماماً للأصلية من حيث الرؤية البشرية.
